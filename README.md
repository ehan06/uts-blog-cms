# Blog CMS - UTS Pemrograman Web

## Deskripsi

Aplikasi Web Blog CMS berbasis PHP dan MySQL dengan fitur CRUD (Create, Read, Update, Delete) untuk mengelola artikel dan kategori.

## Fitur

* Tambah artikel
* Edit artikel
* Hapus artikel
* Manajemen kategori
* Upload gambar

## Teknologi

* PHP Native
* MySQL
* HTML, CSS
* XAMPP

## Cara Menjalankan

1. Import file `db_blog.sql` ke phpMyAdmin
2. Pindahkan folder ke `htdocs`
3. Jalankan Apache dan MySQL di XAMPP
4. Akses melalui browser: `http://localhost/blog`
